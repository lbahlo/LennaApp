DCAF (Database Conversion Analysis Fraud) provides a set of fraud analysis services to customers. Based on customer data, it will analyze and compute (via proprietary algorithms) any fraud associations found on the data. Fraud will be determined via facial recognition technology along with bio metric fraud analysis. Responsiblites on DCAF was the architecture and implementation of the frontend client side of the application. Highlights of this effort beside the SPA architecture implementation is the custom 3D face model viewer and a custom Fraud visualization network graph used to visualize fraud relationships discovered in the analysis phase.
